# Autohouse App

Full-stack vehicle dealership web application built with Node.js, Express, PostgreSQL, and EJS.

## Features

- Vehicle inventory management
- User authentication
- Vehicle detail pages
- Reviews and comments
- PostgreSQL database integration
- Responsive design

## Tech Stack

- Node.js
- Express.js
- PostgreSQL
- EJS
- CSS
- JavaScript
